# How many timeouts does each team get per set in volleyball?

**Short Answer:** Under FIVB rules, each team gets 2 timeouts per set, each lasting 30 seconds. NFHS high school rules allow 2 timeouts per set as well. NCAA grants 2 timeouts per set.

## Full Explanation

Per FIVB Rule 18.1, each team is entitled to a maximum of 2 timeouts per set, each of 30 seconds duration. Unused timeouts do not carry over to the next set. Under USAV domestic rules, each team receives 2 timeouts per set. Under NFHS (high school) rules, each team receives 2 timeouts per set, each 30 seconds. Under NCAA rules, each team receives 2 timeouts per set. In addition, FIVB international matches include 2 Technical Timeouts per set (at points 8 and 16 for the first team to reach those scores), which are not available under NFHS or most domestic club rules.

**Rulesets:** FIVB/USAV/NFHS/NCAA
**Rule Reference:** FIVB Rule 18.1; NFHS Rule 5-5; NCAA Rule 5

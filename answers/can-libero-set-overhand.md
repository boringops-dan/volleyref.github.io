# Can a libero set with overhand finger action in volleyball?

**Short Answer:** The libero may set overhand from behind the attack line, and teammates may attack that set. But if the libero uses an overhand finger set from in front of the attack line, no player on the team may complete an attack hit on that ball above net height.

## Full Explanation

Per FIVB Rule 19.3.1.4, when the libero makes an overhand finger-action set while positioned in front of the attack line (3-meter line), no teammate may attack that ball if it is entirely above the height of the net. This rule prevents the libero from acting as a front-row setter who can enable attacks. If the libero is behind the attack line when setting overhand, there is no such restriction and attackers may legally spike the set. The libero may always use a forearm (bump) set from anywhere on the court. These rules apply under FIVB, USAV, and NCAA. NFHS follows the same overhand set restriction.

**Rulesets:** FIVB/USAV/NFHS/NCAA
**Rule Reference:** FIVB Rule 19.3.1.4

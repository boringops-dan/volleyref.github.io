# Volleyball Scoring Rules

Modern volleyball uses rally scoring (FIVB Rule 6.3): a point is awarded on every rally regardless of which team served.

## Set Scoring

- **Sets 1-4:** First team to 25 points with a 2-point lead wins the set (FIVB Rule 6.4.1). No cap under FIVB/USAV/NCAA. NFHS caps at 27 (NFHS Rule 5-3-1).
- **Deciding set (set 5):** First to 15 points with a 2-point lead (FIVB Rule 6.4.2). No cap under FIVB/USAV/NCAA. NFHS caps at 17.

## Match Format

- Best of 5 sets; first to 3 sets wins the match (FIVB Rule 6.1).
- Some formats use best of 3.

## Timeouts

- **FIVB/USAV/NFHS/NCAA:** 2 timeouts per team per set, each 30 seconds.
- **FIVB international only:** 2 Technical Timeouts per set (at points 8 and 16).

## Service Ace

A service ace is a serve that lands untouched in the opponent's court or cannot be controlled into a playable pass, resulting in a direct point (statistical term; FIVB Rule 12).

VolleyRef vs VolleyWrite.

Both are built for official volleyball scorekeeping. They diverge sharply on platform, price, and who can actually use them. Here's what you need to know before choosing.

Platform Support Pricing Rule Sets Live Scoreboard

TL;DR

VolleyWrite is a mature officiating tool used at NCAA championships, but it only runs on iPad and Windows, costs $129.99/year, and has no spectator sharing. VolleyRef.App runs on any device, costs $50/year (or $5/week), adds FIVB rules and full undo, and lets spectators follow the score live. Choose VolleyWrite if your institution requires it. Choose VolleyRef for everything else.

Detailed Breakdown

Category by Category.

Platform

VR

VolleyRef.App

Runs in any browser. Open app.volleyref.app on an Android, an iPhone, a Windows laptop, a Chromebook, a Mac -- it works. No app store install required. If a tournament provides a device you've never used before, VolleyRef opens in 5 seconds.

VW

VolleyWrite

iPad (iOS) and Windows desktop only. No Android, no iPhone, no Mac, no web. If you're not on Apple tablet or Windows PC, VolleyWrite is not an option for you -- full stop.

Bottom line: If there's any chance you'll need to score on a non-iPad device, VolleyRef is the only option. If your workflow is exclusively iPad-based, both work.

Pricing

VR

VolleyRef.App

First 3 matches free, no card required. Then: $5/week (tournament weekend), $15/month, or $50/year. Your match history stays accessible even if you pause your subscription.

VW

VolleyWrite

$129.99/year for iPad as of July 2024 (up from ~$89.99 previously). Windows is $109.99/year. No weekly option. Historical match data is inaccessible if you don't renew -- a consistent complaint from long-term users.

Bottom line: VolleyRef is $50/year vs VolleyWrite's $129.99 -- 61% less. For tournament-only refs, VolleyRef's $5/week option has no VolleyWrite equivalent.

Rule Enforcement

VR

VolleyRef.App

Enforces FIVB, USAV, NFHS, and NCAA rules automatically. Illegal substitutions blocked. Substitution limits enforced. Libero restrictions tracked per ruleset. Timeout limits enforced. FIVB is the additional ruleset VolleyWrite doesn't cover.

VW

VolleyWrite

Enforces USAV, NFHS, and NCAA rules. Has predictive substitution prompts that auto-suggest the next sub based on roster history -- useful for volunteer scorekeepers who don't know the teams. Does not cover FIVB.

Bottom line: Both cover the main US rulesets well. VolleyRef adds FIVB. VolleyWrite adds predictive sub suggestions. If you ref FIVB events, VolleyRef is the only option. If you work predominantly with volunteers at the scorer's table, VolleyWrite's prompts may reduce errors.

Spectator Scoreboard

VR

VolleyRef.App

One tap generates a shareable URL. Open it on any device -- Android, iPhone, laptop, smart TV. Parents in the stands, coaches, remote viewers, anyone can follow the live score. No app install, no WiFi restriction.

VW

VolleyWrite

Has an iOS Buddy app that can display the score on a second iPad -- but only on the same WiFi network as the scoring device. No web share link, no Android support, requires a second Apple device.

Bottom line: VolleyRef's spectator sharing works anywhere, on any device, with no hardware requirement. VolleyWrite's Buddy app is WiFi-only and Apple-only.

Bottom Line

Who Each App Is Built For.

VR

Use VolleyRef if you...

  • Don't own or want to be tied to an iPad
  • Ref on Android, iPhone, or any other device
  • Ref FIVB-rule events (club, international)
  • Want to share a live score with spectators
  • Want full undo for every action
  • Want to pay $50/year or $5 for a single tournament
  • Don't want to lose access to past matches if you stop paying
VW

Use VolleyWrite if you...

  • Are required to use it by your institution or conference
  • Work exclusively with volunteer scorekeepers who need sub prompts
  • Are on iPad and the $129.99 price is acceptable
  • Need the official NCAA championship track record
  • Only referee USAV/NFHS/NCAA events, never FIVB

Volleyball Scoring Rules by Ruleset.

FIVB, USAV, NFHS, and NCAA all use rally scoring, but the details differ. Substitution limits, timeout rules, and sanctions vary across rulesets. Know which rules apply to your match.

FIVB USAV NFHS NCAA
Rally Scoring Every rally = 1 point
First to 25 Win by 2
Deciding Set to 15 Still win by 2
4 Major Rulesets Different details
The Foundation

Rally Scoring.

Every rally ends with a point. No side-out only scoring. The old system where only the serving team could score is gone from organized volleyball.

Ball Hits the Floor

Ball hits the floor on your side? Point for them. Every single rally results in a point for one team.

Ball Goes Out

Ball goes out of bounds off your team? Point for them. Serving team faults? Point for the receiving team.

Every Rally Counts

Doesnt matter who served. Every rally counts. Win the rally, get the point. Thats it.

Sets and Match Format

How Sets Work Across Rulesets.

The basics are the same everywhere. First to 25, win by 2. But the deciding set and match format have some differences.

Rule FIVB USAV NFHS NCAA
Points to win a set 25 25 25 25
Deciding set points 15 15 25 (3rd set) 25 (5th set)
Win by 2 2 2 2
Point cap None None None None
Match format Best of 5 Best of 5 Best of 5 Best of 5

NFHS plays best of 5, and the deciding 5th set is to 15 in most states. Some states play best of 3 with the 3rd set to 25. Check your state rules.

Substitutions

Substitution Limits.

This is one of the biggest differences between rulesets. The number of subs you get per set changes everything about lineup strategy.

FIVB: 6 per Set

6 substitutions per set per team. Strict. Once youre out of subs, youre done for the set.

USAV: 18 per Set

18 substitutions per set per team. More flexible for lineup changes.

NFHS: 18 per Set

18 substitutions per set per team. Similar to USAV.

NCAA: 15 per Set

15 substitutions per set per team. Unique limit between the others.

Timeouts

Timeout Rules.

Every ruleset gives you 2 timeouts per set. The duration and extras are where they differ.

FIVB

2 timeouts per set, 30 seconds each. Plus automatic technical timeouts at 8 and 16 points (in some competitions).

USAV

2 timeouts per set, 60 seconds each. No technical timeouts.

NFHS

2 timeouts per set, 60 seconds each. No technical timeouts.

NCAA

2 timeouts per set. Media timeouts may apply.

Sanctions

Sanctions and Cards.

Misconduct is handled through a progressive sanction system. The severity escalates.

Yellow Card (Warning).

First offense. No penalty. Just a note. The referee records it and play continues.

Red Card (Penalty).

Loss of serve or point to the opponent. Real consequences now.

Yellow + Red (Expulsion).

Player removed for the rest of the set. Can return next set.

Red + Yellow (Disqualification).

Player removed for the rest of the match. No coming back.

Libero

Libero Rules by Ruleset.

The libero is a defensive specialist in a different jersey. Every major ruleset allows one, but the specifics vary.

What All Rulesets Share

Libero replacements dont count as substitutions. Can only replace back-row players. The libero wears a contrasting jersey so everyone can identify them.

Where They Differ

Specific re-entry timing varies by ruleset. Whether the libero can serve, how many liberos you can designate, and re-designation rules all depend on which rules youre playing under.

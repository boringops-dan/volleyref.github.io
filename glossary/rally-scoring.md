# Rally Scoring - Volleyball Glossary

> Rally scoring is the scoring system used in all levels of modern volleyball. Per FIVB Rule 6.3, a point is awarded to the serving team if the rally is won by the serving team, or to the receiving team if the rally is won by the receiving team. This replaced the old sideout scoring system where only the serving team could score. A set is won by the first team to reach 25 points (15 in the deciding set) with a minimum two-point lead.

**Ruleset:** FIVB
**Rule Reference:** FIVB Rule 6.3

**Related Terms:** sideout

Description of Service

VolleyRef.App is a web-based volleyball scorekeeping application. It allows users to score volleyball matches with automatic rule enforcement for FIVB, USAV, NFHS, and NCAA rulesets. The service is provided as a web application accessible through modern browsers on any device. No installation is required.

Run Your First Match

A referee-first walkthrough of the core flow. This page is based on a field tutorial created by an early tester, rewritten into website docs.

Teams

Start your first match by setting up Home and Away teams.

  • On the first launch screen, tap Start Your First Match.
  • If you have a saved team template, load it. Otherwise create a team.
  • Enter the Home team name, then repeat for the Away team.
What matters The team names and jersey colors drive the scoreboard readability. Keep it simple and correct.

Jersey colors

Set jersey and libero colors so the scoreboard is clear.

  • Pick a preset combo or tap a swatch to customize.
  • Confirm the colors match what is on the court.
  • Use strong contrast between teams when possible.
What matters This is a readability choice. If the colors are wrong, the scoreboard becomes confusing.

Roster

Add players so lineups and substitutions work cleanly.

  • Enter player numbers and names (if you have them).
  • Mark the libero correctly.
  • Double-check duplicates and missing numbers.
What matters Clean rosters prevent downstream errors in lineups and substitutions.

Match rules

Select the ruleset that matches the match you are working.

  • Choose the correct ruleset (USAV, NFHS, NCAA, FIVB) for the event.
  • Confirm any set count or scoring settings if shown.
What matters Ruleset drives substitution limits and other enforcement. Get it right before the first serve.

Side of court

Set which side each team is on to keep the scoreboard aligned with the court.

  • Select the side of court for Home and Away.
  • Confirm it matches what you see on the floor.

First serve

Pick who serves first to start the set correctly.

  • Select the team with first serve.
  • Confirm with the coin toss result.

Team lineups

Enter starting lineups so rotation and substitutions are consistent.

  • Set the starting lineup for each team.
  • Confirm the server position is correct.
  • Save and continue to the scoreboard.

Scoreboard

Score points and keep the match moving without paper overhead.

  • Use the score controls to award points.
  • Use undo and redo when needed.
  • Keep an eye on rotation and libero state as the set progresses.

Libero replacement

Track libero replacements cleanly so the match stays compliant.

  • Use the libero controls to record replacements.
  • Verify the correct player is replaced.

Correcting errors

Mistakes happen. Fix them without tearing up a paper sheet.

  • Use undo for immediate corrections.
  • Use the correction flow when you need to fix a deeper state issue.

Substitutions

Record substitutions quickly and consistently.

  • Select the team and the player coming out.
  • Select the player going in.
  • Confirm the substitution and continue scoring.

Timeouts, cards, score override

Handle timeouts and sanctions without losing the match state.

  • Record timeouts for the correct team.
  • Record cards and sanctions as needed.
  • Only use score override when you must.
What matters Score override should be rare. Use it when reality diverged and you need to correct the official record.

Match display on phone or TV

Share the match display so others can follow along.

  • Open the match display view.
  • Use the share link to show it on a phone or TV.

If you get stuck, start with the match setup flow again. The system is built to keep state consistent. Undo and redo are your friend.
